# jm-mobile

This is a demo application by React Native showing list of items of public api

## Getting Started

[Set up environment](https://reactnative.dev/docs/set-up-your-environment) if required

### Installation

```bash
yarn
```

### Running the Development Server

```bash
yarn web
yarn android
yarn ios
```

Open [http://localhost:8081](http://localhost:8081) to view the application.

### Running Tests

```bash
# Run all tests
yarn test

# Run tests in watch mode
yarn test:watch

# Run tests with coverage report
yarn test:coverage
```

## Key Libraries

- Expo as the base framework
- Redux for state management
- React-hook-form for Form management, Zod for validation
- Jest and React Native Tesing Library for testing
- https://jsoning.com/api as mock product API
